# 应用
## [聊天类/Chatbot](./apps/chatbot/)
常见的需求是：
1. 客服类：从知识库中来获取信息。
2. 陪伴类：虚拟角色/数字分身。

或者是上面两类的综合。

### 个人助理
* [Pi](https://inflection.ai/) I’m your personal AI, designed to be supportive, smart, and there for you anytime. Inflection 做的。
* [copy.ai](https://www.copy.ai/) 给 prompt 做了分类。

### 对话式数据库
