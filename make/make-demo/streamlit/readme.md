# Deploy use [Streamlit](https://streamlit.io/)

Run
```
streamlit run index.py
```

## 注意
当屏幕上有东西要更新时，会重新运行整个文档（和 React 的 function component 很像）。要保持住啊女改，要用 state 来保存之前的状态。
